# Scaling Distributed Applications
Es la capacidad de un sistema para aumentar su rendimiento y disponibilidad mediante la distribución de tareas en múltiples unidades de ejecución, como procesos, hilos o nodos.
# Descipción
El sistema implementa un patrón Supervisor–Worker donde un proceso monitor detecta fallos mediante códigos de salida y aplica una política de recuperación basada en reinicio limitado. Esto garantiza aislamiento, recuperación automática y continuidad parcial del servicio.
